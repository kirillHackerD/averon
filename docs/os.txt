========================================
БИБЛИОТЕКА: os
========================================

ТИП: Стандартная библиотека (встроенная)
ЯЗЫК: Python
ОФИЦИАЛЬНАЯ ДОКУМЕНТАЦИЯ: https://docs.python.org/3/library/os.html

ОПИСАНИЕ:
Предоставляет портативный способ использования функций,
зависящих от операционной системы. Используется для взаимодействия
с файловой системой, управления процессами и переменными окружения.

ОСНОВНЫЕ МОДУЛИ:
- os.path: Манипуляции с путями
- os.environ: Словарь переменных окружения
- os.process: Управление процессами (fork, exec, wait)

ОСНОВНЫЕ ФУНКЦИИ:
- os.getcwd() - Получить текущую рабочую директорию
- os.chdir(path) - Изменить текущую рабочую директорию
- os.listdir(path) - Получить список содержимого директории
- os.mkdir(path) - Создать директорию
- os.remove(path) - Удалить файл
- os.rename(src, dst) - Переименовать файл или директорию
- os.path.join(path, *paths) - Объединить компоненты пути
- os.path.exists(path) - Проверить существование пути
- os.environ.get(key) - Получить переменную окружения
- os.system(command) - Выполнить команду оболочки

УСТАНОВКА:
Установка не требуется. Входит в состав Python.
