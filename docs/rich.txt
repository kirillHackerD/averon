ТИП: Сторонняя библиотека
ЯЗЫК: Python
ОФИЦИАЛЬНАЯ ДОКУМЕНТАЦИЯ: https://rich.readthedocs.io/
РЕПОЗИТОРИЙ: https://github.com/Textualize/rich

ОПИСАНИЕ:
Библиотека для форматированного вывода в терминал.
Поддерживает цвета, стили, таблицы, прогресс-бары, синтаксис, markdown,
деревья, панели и многое другое.

ОСНОВНЫЕ КОМПОНЕНТЫ:
- Console: Основной объект для вывода
- print: Расширенная версия print
- Panel: Панель с рамкой
- Table: Таблица
- Tree: Дерево
- Progress: Прогресс-бар
- Status: Индикатор статуса
- Syntax: Подсветка синтаксиса
- Markdown: Отображение Markdown
- Rule: Разделительная линия
- Layout: Макет

ОСНОВНЫЕ КЛАССЫ:
- Console: Управление выводом в терминал
- Console.print() - Вывод с форматированием
- Console.log() - Логирование с временными метками
- Console.rule() - Разделительная линия
- Console.clear() - Очистка экрана
- Console.status() - Контекстный менеджер статуса
- Console.input() - Ввод с подсказкой

СТИЛИ:
- bold: Жирный
- italic: Курсив
- underline: Подчеркнутый
- blink: Мигающий
- reverse: Инвертированный
- dim: Тусклый
- strike: Зачеркнутый

ЦВЕТА:
- red, green, blue, yellow, cyan, magenta, black, white
- bright_red, bright_green, bright_blue и т.д.
- color(число) - 256 цветов
- rgb(255,128,0) - RGB цвет

        progress.update(task, advance=1)

# ПРОГРЕСС-БАР СТИЛИЗОВАННЫЙ

    SpinnerColumn(),

) as progress:

    time.sleep(2)
