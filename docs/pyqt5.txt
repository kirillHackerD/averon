========================================
БИБЛИОТЕКА: PyQt5
========================================

ТИП: Сторонняя библиотека
ЯЗЫК: Python
ОФИЦИАЛЬНАЯ ДОКУМЕНТАЦИЯ: https://www.riverbankcomputing.com/static/Docs/PyQt5/
ДОКУМЕНТАЦИЯ QT5: https://doc.qt.io/qt-5/

ОПИСАНИЕ:
Официальные Python-привязки (bindings) для фреймворка Qt5.
Мощный инструмент для создания профессиональных кроссплатформенных
графических приложений.

ОСНОВНЫЕ МОДУЛИ:
- QtWidgets: Основные виджеты для GUI
- QtCore: Основные неграфические классы (сигналы, слоты, таймеры)
- QtGui: Компоненты графического интерфейса
- QtNetwork: Сетевые возможности
- QtSql: Работа с базами данных
- QtMultimedia: Работа с мультимедиа

ОСНОВНЫЕ ВИДЖЕТЫ:
- QApplication: Управление приложением
- QMainWindow: Главное окно с меню и панелями
- QWidget: Базовый класс для всех виджетов
- QPushButton: Кнопка
- QLabel: Метка
- QLineEdit: Поле ввода
- QTextEdit: Многострочное текстовое поле
- QTableWidget: Таблица
- QTreeWidget: Дерево
- QComboBox: Выпадающий список

ОСОБЕННОСТИ:
- Сигналы и слоты для связи объектов
- Поддержка Qt Designer для визуального проектирования интерфейсов
- Кроссплатформенность (Windows, macOS, Linux)
- Высокая производительность

УСТАНОВКА:
pip install PyQt5
