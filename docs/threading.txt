========================================
БИБЛИОТЕКА: threading
========================================

ТИП: Стандартная библиотека (встроенная)
ЯЗЫК: Python
ОФИЦИАЛЬНАЯ ДОКУМЕНТАЦИЯ: https://docs.python.org/3/library/threading.html

ОПИСАНИЕ:
Высокоуровневый интерфейс для работы с потоками (threads).
Позволяет выполнять код параллельно с помощью потоков.

ОСНОВНЫЕ КОМПОНЕНТЫ:
- Thread: Объект, представляющий поток выполнения
- Lock: Мьютекс для взаимного исключения
- RLock: Рекурсивная блокировка
- Condition: Условная переменная
- Event: Простая синхронизация потоков
- Semaphore: Ограничение одновременного доступа
- Timer: Выполнение функции с задержкой
- Barrier: Синхронизация нескольких потоков

ОСНОВНЫЕ ФУНКЦИИ И МЕТОДЫ:
- threading.Thread(target=func, args=()) - Создать поток
- thread.start() - Запустить поток
- thread.join() - Ожидать завершения потока
- threading.Lock() - Создать объект блокировки
- lock.acquire() - Захватить блокировку
- lock.release() - Освободить блокировку
- threading.current_thread() - Получить текущий поток
- threading.active_count() - Количество активных потоков

УСТАНОВКА:
Установка не требуется. Входит в состав Python.
