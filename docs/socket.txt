Socket Module Documentation

Low-level networking interface for Python.

Basic Socket Creation:
import socket

# Create a TCP socket
sock = socket.socket(socket.AF_INET, socket.SOCK_STREAM)

# Create a UDP socket
sock = socket.socket(socket.AF_INET, socket.SOCK_DGRAM)

Socket Methods:

# Connect to a server (TCP)
sock.connect(('hostname', port))

# Bind to address (server)
sock.bind(('0.0.0.0', port))

# Listen for connections (TCP server)
sock.listen(backlog)

# Accept connection (TCP server)
conn, addr = sock.accept()

# Send data (TCP)
sock.send(data)
sock.sendall(data)  # Sends all data

# Receive data (TCP)
data = sock.recv(buffer_size)

# Send data (UDP)
sock.sendto(data, (host, port))

# Receive data (UDP)
data, addr = sock.recvfrom(buffer_size)

# Close socket
sock.close()

# Set timeout
sock.settimeout(seconds)

# Check if socket is readable/writable
import select
readable, writable, errored = select.select([sock], [], [], timeout)

Port Scanner Example:
import socket
import concurrent.futures

def check_port(host, port):
    try:
        sock = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
        sock.settimeout(1)
        result = sock.connect_ex((host, port))
        sock.close()
        return port if result == 0 else None
    except:
        return None

# Scan ports
host = '127.0.0.1'
ports = range(1, 1025)

with concurrent.futures.ThreadPoolExecutor(max_workers=100) as executor:
    futures = {executor.submit(check_port, host, port): port for port in ports}
    open_ports = [f.result() for f in concurrent.futures.as_completed(futures) if f.result()]

print(f"Open ports: {open_ports}")

Best Practices:
- Always use socket.close() or 'with' statement
- Set timeouts to avoid blocking
- Use ThreadPoolExecutor for concurrent connections
- Handle exceptions properly
- Use AF_INET6 for IPv6 addresses
