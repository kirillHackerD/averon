========================================
БИБЛИОТЕКА: PyQt6
========================================

ТИП: Сторонняя библиотека
ЯЗЫК: Python
ОФИЦИАЛЬНАЯ ДОКУМЕНТАЦИЯ: https://www.riverbankcomputing.com/static/Docs/PyQt6/
ДОКУМЕНТАЦИЯ QT6: https://doc.qt.io/qt-6/

ОПИСАНИЕ:
Официальные Python-привязки (bindings) для фреймворка Qt6.
Новейшая версия PyQt, основанная на Qt6. Является развитием PyQt5
с улучшенной производительностью и новыми возможностями.

ОСНОВНЫЕ МОДУЛИ:
- QtWidgets: Основные виджеты для GUI
- QtCore: Основные неграфические классы (сигналы, слоты, таймеры)
- QtGui: Компоненты графического интерфейса
- QtNetwork: Сетевые возможности
- QtSql: Работа с базами данных
- QtMultimedia: Работа с мультимедиа
- QtQuick: Интерфейсы на QML

ОСНОВНЫЕ ВИДЖЕТЫ:
- QApplication: Управление приложением
- QMainWindow: Главное окно с меню и панелями
- QWidget: Базовый класс для всех виджетов
- QPushButton: Кнопка
- QLabel: Метка
- QLineEdit: Поле ввода
- QTextEdit: Многострочное текстовое поле
- QTableWidget: Таблица
- QTreeWidget: Дерево
- QComboBox: Выпадающий список

ОТЛИЧИЯ ОТ PYQT5:
- Обновленная система сигналов и слотов (PyQt6.QtCore.pyqtSignal)
- Удалены устаревшие API
- Улучшенная поддержка высоких разрешений (HiDPI)
- Новые возможности Qt6 (модули QtGraphs, QtHttpServer и др.)
- Изменения в именах перечислений (требуется обновление кода)

УСТАНОВКА:
pip install PyQt6

ПРИМЕЧАНИЕ:
PyQt5 и PyQt6 несовместимы друг с другом. Код, написанный для PyQt5,
требует изменений для работы с PyQt6.
