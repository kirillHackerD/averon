========================================
БИБЛИОТЕКА: customtkinter
========================================

ТИП: Сторонняя библиотека
ЯЗЫК: Python
ОФИЦИАЛЬНАЯ ДОКУМЕНТАЦИЯ: https://customtkinter.tomschimansky.com/documentation
РЕПОЗИТОРИЙ: https://github.com/TomSchimansky/CustomTkinter

ОПИСАНИЕ:
Современная библиотека для создания графических интерфейсов (GUI),
является расширением стандартной библиотеки tkinter. Предоставляет
современные виджеты с поддержкой тем (светлая/темная) и кастомизации.

ОСНОВНЫЕ ВИДЖЕТЫ:
- CTk: Главное окно приложения
- CTkButton: Кнопка
- CTkLabel: Метка (текст)
- CTkEntry: Поле ввода
- CTkTextbox: Текстовое поле
- CTkFrame: Контейнер для группировки виджетов
- CTkScrollbar: Полоса прокрутки
- CTkSwitch: Переключатель
- CTkCheckBox: Флажок
- CTkRadioButton: Радиокнопка
- CTkSlider: Слайдер
- CTkProgressBar: Индикатор прогресса
- CTkComboBox: Выпадающий список
- CTkOptionMenu: Меню выбора

ОСНОВНЫЕ ВОЗМОЖНОСТИ:
- appearance_mode: Установка темы ("Dark", "Light", "System")
- set_default_color_theme: Установка цветовой схемы
- масштабирование виджетов под разные разрешения экрана
- поддержка современных шрифтов и анимаций

УСТАНОВКА:
pip install customtkinter

ТРЕБОВАНИЯ:
Требует установленную библиотеку tkinter (входит в стандартный Python)
